# Changelog

The ForVoyez API is continuously evolving to provide you with new features, enhancements, and improvements. This changelog details the latest changes, updates, and bug fixes to help you stay informed about the API's development.

## Version 1.0.0 (2024-05-20)
This is the initial release of the ForVoyez API. The following features are included in this version:
- Image description generation
- Custom context and JSON schema support
- API authentication with API keys
- API response caching
- Usage statistics and analytics
- Subscription plans and billing
- User account management
- Comprehensive API documentation
- Code examples and tutorials
- Support and contact information
- Social media links
- Playground for testing API requests
- Changelog for tracking updates

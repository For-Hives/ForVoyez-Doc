# Changelog

The ForVoyez API is continuously evolving to provide you with new features, enhancements, and improvements. This changelog details the latest changes, updates, and bug fixes to help you stay informed about the API's development.

## Version 1.1.0 (2024-09-20)

This update introduces multilingual support and revises our pricing structure. The following changes are included in this version:

- Added multilingual support for metadata generation
  - New `language` parameter in API requests to specify the desired output language
  - Updated documentation to reflect the new language feature
  - Modified examples in various programming languages to demonstrate language usage
- Revised pricing structure and credit system
  - Updated subscription plans with new credit limits
  - Adjusted credit pack options for additional purchases
- Enhanced error handling
  - Added new error code for invalid language requests
- Updated API documentation
  - Reflected changes in endpoints, parameters, and response formats
  - Added examples for multilingual requests and responses

## Version 1.0.0 (2024-05-20)

- Image description generation
- Custom context and JSON schema support
- API authentication with API keys

- Usage statistics and analytics
- Subscription plans and billing
- User account management
- Comprehensive API documentation
- Code examples and tutorials
- Support and contact information

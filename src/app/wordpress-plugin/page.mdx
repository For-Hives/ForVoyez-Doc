# WordPress Plugin: ForVoyez Auto Alt Text for Images

## Introduction

ForVoyez Auto Alt Text for Images is a powerful WordPress plugin that uses artificial intelligence to automatically generate SEO-optimized alternative texts (alt text) for your images. This plugin enhances your website's accessibility and search engine optimization effortlessly.

## Installation

1. Download the plugin zip file on GitHub (or through the workplace plugin marketplace)

```sh
https://github.com/For-Hives/ForVoyez-Wordpress-plugin/releases/
```

2. Activate the plugin through the WordPress admin interface. (drop the zip file on the plugin page in your wordpress)

## Configuration

1. Go to the ForVoyez Auto Alt Text settings page in your WordPress admin area.

2. Enter your ForVoyez API key.

<ScreenConfiguration />

To obtain an API key if you haven't one:

- Visit the [ForVoyez signup page](https://forvoyez.com/signup) to create an account.
- Once logged in, navigate to your [ForVoyez dashboard](https://forvoyez.com/dashboard).
- Generate an API key in the API section of your dashboard.
- Copy the API key and paste it into the provided field.

3. Click "Save API Key" to store your settings.

4. Configure any additional settings according to your preferences.

## Usage

### Dashboard

The dashboard gives you a quick overview of your image status:

<ScreenDashboard />

- **Total Images**: Total number of images in your media library.
- **Images Completed**: Number of images that have generated alt text.
- **Images Without alt/caption**: Number of images still requiring processing.

### Managing Images

1. Go to the "Manage Images" tab in the plugin interface.

<ScreenManage />

2. You'll see a grid view of your images with visual indicators for missing metadata.

3. To process images:

- Select the images you want to process.
- Choose "Generate Alt Text" from the bulk actions dropdown.
- Click "Apply" to start the process.

4. Once processing is complete, you can view the results

The plugin will generate high-quality titles, alt texts, and captions for a variety of image types.

5. You can manually edit any generated text as needed.

### Automatic Processing

The plugin will automatically process new images as they are uploaded. You can adjust this behavior in the plugin settings.
